- 📄 SvelteKit File Icons and Scaffolding
- 📒 Templates (TypeScript, Tailwind, Mdsvex)
- 🧹 Formatting
- ⌨️ Vim Keybindings
- 👻 Hide Config Files
- 🎨 Command Palette
